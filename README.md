# Aknova - Site Web Institutionnel

Site web officiel d'Aknova, bureau d'études environnementales au Bénin.

## Description

Site web vitrine présentant les services, l'expertise et les réalisations d'Aknova dans le domaine des études environnementales.

## Technologies utilisées

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- AOS (Animate On Scroll)
- Autres librairies : Swiper, GLightbox, etc.

## Structure du projet

```
frontend/
├── assets/
│   ├── css/
│   ├── img/
│   ├── js/
│   ├── scss/
│   └── vendor/
├── forms/
└── *.html
```

## Installation locale

1. Cloner le repository
```bash
git clone [url-du-repo]
```

2. Ouvrir le fichier `index.html` dans un navigateur web
   - Ou utiliser un serveur local comme Live Server dans VS Code

## Déploiement

Ce site est déployé sur Netlify.

[![Netlify Status](https://api.netlify.com/api/v1/badges/votre-badge-id/deploy-status)](https://app.netlify.com/sites/aknova/deploys)

### Configuration Netlify

- Directory de build : `frontend`
- Command de build : none
- Directory de publication : `frontend`

## Maintenance

Pour mettre à jour le site :
1. Modifier les fichiers nécessaires
2. Commiter les changements
3. Pousser vers la branche main
4. Le déploiement se fera automatiquement sur Netlify
